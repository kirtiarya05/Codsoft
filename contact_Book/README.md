# Contact Manager Pro

A sleek, professional contact management system built with Python and Tkinter. This application allows you to efficiently store, search, and manage your connections. Built for the **Codsoft Internship - Task 5**.

## 🌟 Key Features
- **Dynamic Search**: Instantly filter contacts by Name or Phone Number using the integrated search bar.
- **Full CRUD Support**: Add, View, Update, and Delete contacts with ease.
- **Persistent Storage**: All contacts are automatically saved to `contacts.json`.
- **Modern Aesthetic**: Dark-themed user interface matching the professional suite of tools developed in previous tasks.
- **Detailed Information**: Store names, phone numbers, emails, and physical addresses.

## 🛠️ Installation

1. Navigate to the project directory:
   ```bash
   cd Task5_ContactBook
   ```
2. Run the application:
   ```bash
   python main.py
   ```

## 📋 Usage
- **Add**: Fill in the details on the right and click "ADD CONTACT".
- **Search**: Type any name or number in the top-left search bar.
- **Update**: Select a contact from the list, modify their details, and click "UPDATE CHANGES".
- **Delete**: Select a contact and click "DELETE CONTACT".
