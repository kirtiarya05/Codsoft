# Modern Scientific Desktop Calculator 🧮

A sleek, dark-themed **Scientific Calculator** built entirely in **Pure Python**. This version features a modern desktop GUI using the `tkinter` library, providing a high-performance and lightweight experience without the need for a web browser.

![Desktop Calculator Hero](https://images.unsplash.com/photo-1587145820266-a5951ee6f620?auto=format&fit=crop&q=80&w=1000)

## ✨ Features

- 🌌 **Modern Dark UI:** Professional aesthetic with a deep navy and purple accent theme.
- 🧪 **Scientific Capabilities:** Supports Trigonometry (sin, cos, tan), Logarithms, Square Roots, Powers (xʸ), and constants (π, e).
- 📜 **Live History:** Displays your current expression and previous calculation in real-time.
- ⌨️ **Keyboard Support:** Full numerical and operator support directly from your keyboard.
- 🚀 **Zero Dependencies:** Built using standard Python libraries only.
- 🪟 **Desktop Native:** Runs as a standalone window on Windows, macOS, and Linux.

## 🛠️ Built With

- **Language:** Python 3.x
- **GUI Library:** Tkinter (Standard Library)
- **Math Engine:** Python `math` module

## 🚀 Quick Start

### Prerequisites

- Python 3.x installed on your system.

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/kirtiarya05/codesoft.git
   cd codesoft/Task2_Calculator
   ```

2. **No Dependencies Needed!**
   Since this project uses only standard libraries, you don't need to install anything via `pip`.

### Running the App

Simply run the `main.py` file:
```bash
python main.py
```

## ⌨️ Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `0-9` | Numbers |
| `+`, `-`, `*`, `/` | Basic Operators |
| `.` | Decimal Point |
| `Enter` | Calculate Result |
| `Backspace` | Delete Last Character |
| `Escape` | Clear All (AC) |
